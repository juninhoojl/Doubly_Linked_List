# Doubly Linked List (Implemented in C)

![Drag Racing](DoublyLinkedList.png)

## Struct Node (node)
```c
typedef struct Node {
    int data;
    struct Node * next;
    struct Node * prev;
}node;

```

## Struct Hnode (hnode)

```c
typedef struct Hnode {
    int tam;
    struct Node * primeiro;
    struct Node * ultimo;
}hnode;

```
